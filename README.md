# UberAnalysis

Analyzed Uber trip data to identify **peak ride demand**, popular pickup locations, and time-based ride patterns.

## Insights and Work Done
- Identified peak hours: **8–10 AM & 5–8 PM**
- Ride frequency distribution by pickup location
- Heatmap visualization of demand across time and location

## Technologies used
- Python, Pandas, Numpy
- Matplotlib, Seaborn
- Jupyter notebook
